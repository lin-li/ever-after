A) FAQs - top

How to Change the logo?

To change the logo, you need to find(CTRL+F) the word "logo" in the index.html file. Replace it with an image or text.

How to Change the heart icon, social icons?

I am using Font Awesome for the icons, you can copy the classes and replace it with the template.

How to change the background images in the header?
  • Go to js folder and edit main.js
  • Find this text "Home Background slider"
  • Underneath it, you will find the images for the slider.
  • You can add more images by copying another line.
Change the date in the countdown?
  • Go to js folder, edit main.js
  • As you open the file, you will see the script for the countdown
    • Save the file
Change the images in Love story section?
  • First, let's edit the index.html
  • Required image size for the bride and groom are 220x220
  • Required image size for the couple are 250x250
Change the images in Gallery section
  • First, let's edit the index.html
  • Find this text in index.html: "photostack"
  • The preferable thumbnail size is 240x240 and it's up to you for the larger preview which shows up in the modalbox.
  • To change the thumbnail - replace the location on what is inside the "img src"
  • To change the large preview, replace on what is inside the "a href"
  • As you can see below there is a figure tag with data-dummy - these are the images that will be displayed on the sides but not on the bullets
How to add your email in the RSVP?
  • Go to process.php
  • Find first the falconerie.04@gmail.com
  • Replace it with the email of your choice.
  • Save the file.

B) Files and Structure - top

HTML

I am using the latest Bootstrap 3 for the theme's HTML structure.

CSS

Css files are located under the css folder

  • bootstrap.min.css I'm using two CSS files in this theme. The first one is a Bootstrap file that includes normalize.css . This makes browsers render all elements more consistently and in line with modern standards. It precisely targets only the styles that need normalizing.
  • main.css contains all of the specific stylings for the template.
  • font-awesome.min.css - this is used for the icons that are all used in the site. Check it out in
  • magnific-popup.css - used for modalboxes.
  • supersized.css - used for header slider.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

JavaScript

This theme imports 14 Javascript files.

  1. jQuery
  2. Bootstrap.js
  3. Modernizr.js
  4. HTML5shiv.js
  5. ajax-mail.js
  6. main.js - custom functions for the template.
  7. jquery.lwtCountdown-1.0.js
  8. classie.js - for the portfolio
  9. SmoothScroll.js
  10. supersized.3.2.7.min.js
  11. jquery.smooth-scroll.js - menu scroller
  12. isotopes.js
  13. jquery.hoverdir.js
  14. jquery.magnific-popup.min.js
